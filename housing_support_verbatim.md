# 주택지원

## 주택지원사업이란?
태양광, 태양열, 지열, 소형풍력, 연료전지 등의 신·재생에너지원을 주택에 설치할 경우 설치비의 일부를 정부가 보조지원하는 사업입니다.

## 지원대상
| 신청대상 | 단독주택 / 공동주택 |
| :--- | :--- |
| **단독주택** | 기존 또는 신축 주택의 소유자 또는 소유 예정자 |
| **기존 공동주택** | 공동주택 소유자 또는 입주자 대표(등)<br>*입주자(세대주 전체) 자필 동의서 또는 입주자 대표회의 의결내역 제출 필수 |
| **신축 공동주택** | 신축 중인 공동주택의 시행·공사 대표 또는 입주자 대표 등<br>*설치완료기한내 설치완료가 가능한 신축 공동주택을 포함 |
