# 건물지원

## 건물지원사업이란?
주택을 제외한 모든 일반건물에 대하여 태양광발전소 설치 시 설치비용의 50% 내외를 지원받고 설치하는 전기요금을 획기적으로 절약할 수 있는 태양광 보조 정부지원사업입니다.

## 전기요금 절감효과
| 구분 | 내용 |
| :--- | :--- |
| **일반** | 일반모듈 kw당 972,000원 / 저탄소모듈 kw당 1,009,000원 |
| **축사** | 일반모듈 kw당 1,120,000원 / 저탄소모듈 kw당 1,167,000원 |
*설치 여건에 따라 사업비용이 측정되며 최종 자부담 금액은 달라질 수 있습니다.
**→ 최대 200kw까지 지원**

### 전기요금 절감효과
| 태양광 설치용량 | 설치 후 요금 최대 절감액(연간) |
| :--- | :--- |
| 18KW | 2,536,390원 절감 |
| 50KW | 7,045,528원 절감 |
| 100KW | 14,091,055원 절감 |
| 200KW | 28,182,110원 절감 |
*태양광 설치 용량 및 일조량 등에 따른 차이가 있으며, 평균 일 3.6시간 발전 기준으로 전기요금 변동여부에 따라 달라질 수 있습니다.
